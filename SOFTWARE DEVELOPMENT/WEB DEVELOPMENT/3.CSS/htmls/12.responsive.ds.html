Responsive web design

Las mediaqueries son un mecanismo que tiene css para controlar el como se van a ver los elementos html en función del tamaña que vaya teniendo la pantalla. Basicamente permiten modificar el aspecto en función del tamaño del dispositivo que abre la pagina

Las media queries es donde establecemos estilos en función de ciertas condiciones como por ejemplo el tamaño de la pantalla. Alog asi como si el tamaño de la pantalla es de 700 x 800 entonces mostrar los estilos asi

Técnicas de diseño adaptable

Maquetación mobile first

Se basa en maquetar primero pensando el tamaños para telefonos para luego a partir de esto adaptar a desktop. Esta opción se considera la mejor debido a que es mas amigable y facil adaptar de mobile a desktop

¿Y como se hace esto?, pues con las media-queries se pone las condiciones con tamaños que tienen la siguiente logica: Cuando el tamaño de la pantalla supere x tamaño cambiar los estilos

Maquetación deskto first

Se basa en maquetar primero pensando el tamaños para escritorio para luego a partir de esto adaptar a moviles
